Keterangan:
Chatbot bahasa native php dengan menggunakan model AI  NLP/LLM untuk memproses bahasa alami dan memberikan respon yang relevan kepada pengguna.
● Use Case: Travel Assistant
● Contoh Parameter Kreatif: gaya bahasa (formal/santai), integrasi API eksternal, atau fitur tambahan seperti memory dan rekomendasi.

Cara Installasi:
1. Install xampp/laragon
2. Copy File di Folder htdocs/www
3. Jalankan http://localhost/chatbot/
